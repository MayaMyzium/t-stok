改良型多空比信號

如果您想進一步運用複雜模型分析永續合約做多/做空點位，本專案提供 advanced_analysis.py 腳本。該腳本實現了一套可落地交易的公式： 先對多空比（LSR）進行 winsorize 及滾動 z 分數處理，再結合變化率、成交量 z 分數、價格動能與 ATR、自動分位門檻與趨勢過濾，最後產生綜合信號 S^* 以及自適應進出場點位。建議在 1小時或 15 分鐘的 K 線資料上使用，並至少保留 1000 根歷史資料作為分位數計算參考。

您可以將自己的資料（時間、open、high、low、close、volume、lsr）整理成 CSV，再使用 Python 執行以下範例計算信號與進出場價位：

import pandas as pd
from advanced_analysis import compute_signals

df = pd.read_csv('your_data.csv')
signals = compute_signals(
    price=df['close'].values,
    high=df['high'].values,
    low=df['low'].values,
    lsr=df['lsr'].values,
    volume=df['volume'].values
)
print(signals.tail())

程式會回傳一個 DataFrame，其中包含信號、進場價（entry_long/entry_short）、止損點（sl_long/sl_short）與建議部位大小等欄位，可作為策略參考。請注意，示例僅供學習用途，不構成投資建議。
